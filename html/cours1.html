0. Hello world

Si vous avez déja fait de la programmation dans un autre langage alors que vous avez déja vu cette fameuse expression Hello World , Bonjour tout le monde en français, dans cette étape on affiche un petit message à l'écrant bonjour tout le monde grace à la fonction print()

Example 0

#affichage du message hello world

print("Hello world")

1. Un monde de variables

Le but d'un programme est de faire quelque chose avec des données ou, en d'autres termes, avec les choses que vous mettez dans votre programme. Souvent, votre programme utilisera une ou plusieurs variables pour manipuler les données,ces derniéres sont caracrérisé en plusieurs types, selon la nature de l'information a stocké s'il s'agit d'un nom de personne, prénom , ect il est préférable d'utiliser une variable de type chaine de caractéres (string),s'il s'agit de l'age d'une personne année de naissance , il est préférable d'utiliser un nombre naturel(int) , s'il s'agit de la moyenne d'un ensemble de valeurs alors on doit utiliser une valeur réelle (float) .

Exemple 1

#déclaration d'une variable de type string

nom = "mohamed"

#déclaration d'une variable de type int

age = 21

#déclaration d'une variable de type float

moyenne = 10.25

2. L'entrée du clavier

Les développeurs ont souvent besoin d'interagir avec les utilisateurs, soit pour obtenir des données, soit pour fournir une sorte de résultat. La plupart des programmes utilisent aujourd'hui une boîte de dialogue pour demander à l'utilisateur de fournir un certain type d'entrée. Alors que Python nous fournit la fonction input pour lire les entrées du clavier.

Example 2

#lecture d'une valuer saisie par l'utilisateur

age = input("donner votre age")

Note: le type de la variable lu par l'entrée du clavier est String

3. Les conditions

Cette notion est l'une des plus importante en programmation. L'idée est de dire que si telle variable a telle valeur alors faire cela sinon cela.

Example 3.1

if - else
consiste à vérifier si une condition alors action sinon une autre action

#vérification de l'age de l'utilisateur

if ( age > 18 ):

print("vous êtes majeur")

else:

print("vous êtes mineur" )

Example 3.2

if - elif - else

consiste à vérifier si une condition alors action sinon si une autre condition alors action sinon une autre action

#vérification de l'age de l'utilisateur

if ( age > 18 ):

   print("vous êtes majeur")

elif ( age < 10):

   print("vous êtes gamin" )

else :

   print("vous êtes mineur")
