8. Les dictionnaires

Un dictionnaire en python est une sorte de liste mais au lieu d'utiliser des index , on utilise des clés alphanumériques, on les appelles aussi les tableaux associatifs.

Example 8.1

#la déclaration d'un dictionnaire se fait comme suit

a = dict ()

Pour ajouter des valeurs à un dictionnaire il faut indiquer une clé ainsi qu'une valeur:

Example 8.2

#l'ajout de l'element date de naissance

a[ "date de naissance" ] = "1999"

Example 8.3

Il est possible de supprimer une entrée en indiquant sa clé
#la suppression de l'element date de naissance

del   a [ "date de naissance" ]

9. Les exceptions

En python, une exception est une instance d'une classe héritée de BaseException. Dans un bloc try, la clause except traite non seulement la classe d'exception qu'elle mentionne, mais aussi toutes les classes dérivées de cette classe (contrairement à ses classes mères). Deux classes qui ne sont pas liées par héritage ne sont jamais équivalentes, même si elles ont le même nom.
En général les exceptions sont utilisé pour la gestion des erreurs .

Example 9

#dans cette example on essaie de diviser par 0

try :

    print( 10 / 0 )

catch :

    print("vous pouvez pas déviser par 0")

Une exception sera levée avec le message Suivant "ZeroDivisionError: division by zero "

10. Gestion des fichiers

La gestion des fichiers est une partie indispensable dans chaque langage de programmation ,pour cela python propose de nombreuse fonctions et méthodes pour la gestion des fichiers.

Example 10.1

#ouverture d'un fichier en mode lecture

fichier = open ("data.txt", "r")

#lecture du contenu du fichier

print (fichier.read())

Example 10.2

#ouverture d'un fichier en mode écriture

fichier = open ("data.txt", "w")

#écriture sur le fichier

fichier.write("hello world" )
